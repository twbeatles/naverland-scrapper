# **🏠 네이버 부동산 매물 크롤러 Pro Plus (v13.0)**

네이버 부동산의 아파트 매물 정보를 수집, 분석, 관리할 수 있는 강력한 데스크톱 애플리케이션입니다.  
다중 단지 동시 크롤링, 실시간 가격 변동 추적, 시세 분석, 신규 매물 알림, 그리고 엑셀 내보내기 기능 등을 제공합니다.

Claude Opus 4.5, Gemini 3.0 Pro 를 이용하여 제작 및 지속 개선중입니다.

## **✨ 주요 기능**

### **v13.0 신규 기능 (Analytics & Stability)**

* **📊 시세 분석 대시보드:** 통계 카드, 거래유형별 파이차트, 가격대별 히스토그램으로 수집 데이터를 한눈에 분석
* **📈 트렌드 분석:** 주간/월간 가격 추이, 상승/하락/횡보 트렌드 자동 분석
* **🃏 카드 뷰 모드:** 테이블 외에 시각적인 카드 형태로 매물을 조회할 수 있는 뷰 모드 추가
* **⭐ 즐겨찾기 탭:** 관심 매물을 즐겨찾기로 등록하고 별도 탭에서 메모와 함께 관리
* **🔄 안정성 강화:** 네트워크 오류 시 자동 재시도, Rate Limit 감지 및 지수 백오프 대기
* **🕐 최근 본 매물:** 조회한 매물 히스토리 자동 저장 및 조회 기능

### **v12.0 기능**

* **💰 평당가 계산:** 매물별 평당가 자동 계산 및 정렬
* **📝 매물 메모:** 개별 매물에 메모 작성 및 관리
* **🗄️ 크롤링 캐시:** 동일 단지 재수집 시 캐시 활용으로 속도 향상
* **👻 매물 소멸 추적:** 사라진 매물 자동 감지 및 표시

### **v11.0 이전 기능**

* **🎨 현대적인 UI:** Glassmorphism 효과, HiDPI 지원, Toast 알림
* **🌗 다크/라이트 모드:** 눈의 피로를 줄이는 테마 전환
* **📉 가격 변동 추적:** 이전 기록과 비교하여 가격 상승/하락 하이라이트
* **🆕 신규 매물 배지:** 새로 등록된 매물 'NEW' 배지 표시
* **🔍 고급 필터링:** 가격 범위, 면적, 층수, 키워드 상세 조건 필터
* **📊 엑셀 템플릿:** 원하는 컬럼만 선택하여 내보내기

### **핵심 기능**

* **다중 단지 수집:** 여러 아파트 단지를 등록하고 한 번에 크롤링  
* **데이터베이스 관리:** 로컬 DB(sqlite3)에 안전하게 저장  
* **그룹 관리:** 관심 단지를 그룹별로 묶어 관리  
* **알림 시스템:** 조건에 맞는 매물 발견 시 알림  
* **편의 기능:** 트레이 최소화, 예상 소요 시간 표시, 결과 대시보드

## **🛠 설치 및 실행 방법**

### **1\. 필수 요구 사항**

* Python 3.8 이상  
* Google Chrome 브라우저 (최신 버전 권장)

### **2\. 라이브러리 설치**

터미널(CMD)에서 아래 명령어를 입력하여 필수 라이브러리를 설치하세요.  
pip install PyQt6 undetected-chromedriver beautifulsoup4 openpyxl plyer matplotlib

### **3\. 프로그램 실행**

python "부동산 매물 크롤러 v13.0.py"

## **📖 사용 가이드**

1. **단지 등록**  
   * **검색:** '단지 목록' 탭에서 단지명과 단지 ID(네이버 URL의 숫자 부분)를 입력하여 추가합니다.  
   * **일괄 등록:** 'URL등록' 버튼을 눌러 여러 URL을 붙여넣으면 자동으로 파싱하여 등록합니다.  
2. **조건 설정**  
   * 거래 유형(매매/전세/월세)을 선택합니다.  
   * 필요 시 가격 및 면적 필터를 활성화하여 범위를 지정합니다.  
3. **크롤링 시작**  
   * ▶️ 크롤링 시작 버튼을 누르거나 단축키 Ctrl+R을 입력합니다.  
   * 진행 상황과 예상 남은 시간이 하단에 표시됩니다.  
4. **결과 확인 및 저장**  
   * 수집된 매물은 테이블에 표시되며, 더블 클릭 시 네이버 부동산 페이지로 이동합니다.  
   * 💾 저장 버튼을 눌러 엑셀(xlsx), CSV, JSON 형식으로 데이터를 내보낼 수 있습니다.

## **📂 파일 구조**

프로그램 실행 시 자동으로 생성되는 디렉토리입니다.

* data/: 데이터베이스(complexes.db), 설정 파일(settings.json), 히스토리 등이 저장됩니다.  
* logs/: 날짜별 실행 로그 파일이 저장되어 오류 추적에 사용됩니다.

## **🔨 빌드 (실행 파일 생성)**

PyInstaller를 사용하여 단독 실행 파일(.exe)을 생성할 수 있습니다.

### **빌드 요구 사항**
```bash
pip install pyinstaller
```

### **빌드 명령**
```bash
pyinstaller "부동산_매물_크롤러_v13.0.spec"
```

빌드 완료 후 `dist/네이버부동산크롤러_v13.0.exe` 파일이 생성됩니다.
